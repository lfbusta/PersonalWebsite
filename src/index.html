Shardcraft

Shardcraft is a small private whitelisted vanilla minecraft server that a friend and I have been hosting for around four years now. After all these years I think its about time I made a website for it.


Features:

• Server Status • Player Status
• Dynamic Map • Player Statistics
• Dynamic Weather Forecast
